This is a free software alternative discord client.

# ATTENTION
This could be referenced as `Self-botting` and is suspect to vialation of Discords ToS **USE AT OWN RISK**, no binaries will be provided.

## Running locally 
### Prerequisites
* A JDK 12
### Running
Windows
```cmd
> gradlew run
```
mac or gnu/linux
```sh
$ ./gradlew run
```

### Building FAT JAR
Windows
```cmd
> gradlew shadowJar
```
mac or gnu/linux
```sh
$ ./gradlew shadowJar
```
